Skins¶

Ren'Py 支持为启动器（launcher）更换皮肤——也就是说，您可以改变启动器的外表。 为了达到这个目的，请按照如下步骤操作：

皮肤仅限于当前 Ren'Py 版本, 而且不能向前或者向后兼容。

  1. 在启动器中打开启动器工程。 您可以通过进入设置界面（preferences screen）来完成它。

  2. 创建 skin.rpy 脚本文件。

  3. 拷贝如下代码到 skin.rpy 之中：

    init python:

    # 非交互式文字的颜色
    TEXT = "#545454"

    # 按钮不同状态的颜色
    IDLE = "#42637b"
    HOVER = "#d86b45"
    DISABLED = "#808080"

    # 相反的文字按钮（reversed text buttons）的颜色 （选择列表条目）。
    REVERSE_IDLE = "#78a5c5"
    REVERSE_HOVER = "#d86b45"
    REVERSE_TEXT = "#ffffff"

    # Colors for the scrollbar thumb.
    SCROLLBAR_IDLE = "#dfdfdf"
    SCROLLBAR_HOVER = "#d86b45"

    # 一个用作分隔图案的图片.
    PATTERN = "images/pattern.png"

    # 一个可以当做背景的可显示对象（displayable）
    BACKGROUND = "images/background.png"

    # 一个用作窗口背景的可显示对象
    # 包含命令（commands），参数选择（preferences）和导航信息（navigation info）。
    WINDOW = Frame("images/window.png", 0, 0, tile=True)

    # 一个用作工程列表（projects list）背景的可显示对象。
    PROJECTS_WINDOW = Null()

    # 一个用作信息框（information boxes）背景的可显示对象
    INFO_WINDOW = "#f9f9f9"

    # 信息框的标题颜色
    ERROR_COLOR = "#d15353"
    INFO_COLOR = "#545454"
    INTERACTION_COLOR = "#d19753"
    QUESTION_COLOR = "#d19753"

    # 输入文字的颜色
    INPUT_COLOR = "#d86b45"
  1. 编辑 skin.rpy 来为启动器换皮肤，请把您所用到的所有图片文件放到启动器的游戏目录。

一个不正确的 skin.rpy 可能导致启动器无法正常启动。 为了修复这个错误，您你需要从启动器的目录中移除skin.rpy和skin.rpyc这两个文件, 打开启动器，然后再把它们放回去。
